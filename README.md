## Installation and Running

### Requirements
- Node should be installed 
- A package manager i.e yarn or npm

### Clone the repository
```
$ git clone https://github.com/hoslack/sms_manager.git
$ cd sms_manager 
```

## Installing dependencies

```
$ yarn install
```

or

```
$ npm install
```

## Running the application

```
$ yarn start
```

or 

```
$ npm start
```

## Running the tests

```
$ npm test
```

or

```
$ yarn test

```
